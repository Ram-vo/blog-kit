# Code of Conduct

## Our Standard

This project aims for rigorous, respectful technical collaboration.

Participants are expected to bring:

- direct, fact-based technical discussion
- respect for different levels of experience
- focus on solving concrete problems
- clear communication in issues, discussions, and PRs

Personal attacks, harassment, discrimination, and abusive behavior are not tolerated.

## Enforcement

Maintainers may moderate, edit, or close contributions that violate these standards.
